CIT（シーアイティ） - イタリアン レストラン サイト

このフォルダには、授業用に作ったシンプルな多ページWebサイトがあります。

構成
- index.html — トップページ
- pages/menu.html — メニュー
- pages/about.html — お店について
- pages/contact.html — アクセス / 予約（デモフォーム）
- css/style.css — スタイル
- images/logo.svg — ロゴ（SVG）

このサイトは、フォルダ `素材/` にアップロード済みの写真を直接参照して表示するように構成しています。
以下の素材ファイルをページ内で使用しています（主要な例）：

- ワインと人気メニュー.jpg — トップのヒーロー
- 国産牛ステーキ.jpg — ギャラリー、メニュー
- トマトとモッツァレラチーズ.jpg — ギャラリー
- フレッシュなサラダ.jpg — ギャラリー
- 生ハム.jpg — ギャラリー
- ワイン.jpg / ワイン2.jpg / ワイン3.jpg — ワイン関連のビジュアル
- こだわり1絶品パスタ.jpg — メニュー / カード
- こだわり5チーズ料理.jpg — チーズ関連のカード
- こだわり3コース料理.jpg — コースのイメージ
- 内装1.jpg / 内装2.jpg / 外観.jpg — お店の雰囲気（Aboutページ）

そのため、サイトを表示する際はプロジェクトの親フォルダ (`d:/中川研/restaurant`) をルートにしてサーバーを起動するのが簡単です。

使い方
1. エクスプローラーで `d:/中川研/restaurant/website/index.html` をダブルクリックしてブラウザで開くか、ローカルサーバーで公開してください。
2. このサイトはあなたがアップロード済みの `素材/` フォルダ内の写真を参照するように設定しています。サイト内で参照していない要素（SVGプレースホルダ等）は削除済みで、各ページのビジュアルは `素材/` の実画像に合わせて作り直しています。簡単に写真を表示するには以下のいずれかを行ってください。

	推奨 (簡単): プロジェクトの親フォルダをルートにしてサーバーを起動します。

	```bash
	# 親フォルダ (d:/中川研/restaurant) をルートにする例
	cd /mnt/d/中川研/restaurant
	python -m http.server 8000
	# ブラウザで http://localhost:8000/website/ を開く
	```

	代替: 画像を `website/images/` にコピーして HTML の img src を `images/ファイル名.jpg` に書き換えてください。

3. 実画像は大きさを 800〜1600px 幅にリサイズして最適化することをおすすめします。
